                Comment créer une règle
                
            

            

                

                    
1 Nom de la règle

                    
Donnez un nom court et descriptif (ex. Administratif, Enseignants CDI). Ce nom sera utilisé comme préfixe dans les noms de fichiers CSV générés.

                

                

                    
2 Règle active / inactive

                    
Une règle inactive ne peut pas générer de CSV. Elle reste visible dans la liste (section séparée, grisée) mais est suspendue. Utilisez ce toggle pour désactiver temporairement sans supprimer.

                

                

                    
3 Niveau de groupement

                    

                        

                            1
                            Global
                            1 fichier CSV contenant tous les utilisateurs correspondants.
                        

                        

                            2
                            Par DO
                            1 CSV par direction (Department) + 1 CSV global. Idéal pour des listes par service.
                        

                        

                            3
                            Par centre
                            1 CSV par centre (Office) dans chaque DO + 1 CSV par DO + 1 CSV global.
                        

                    

                

                

                    
4 Conditions d'inclusion

                    
Définissez les critères que les utilisateurs doivent satisfaire pour être inclus. Si plusieurs conditions sont ajoutées, elles fonctionnent en OU : un utilisateur est inclus s'il correspond à au moins une condition.

                

                

                    
5 Conditions d'exclusion

                    
Parmi les utilisateurs inclus, retirez ceux qui correspondent à l'une des conditions d'exclusion. Comme pour l'inclusion, les exclusions fonctionnent en OU.

                

                

                    
6 Opérateurs disponibles

                    

                        
est exactement — correspondance exacte (insensible à la casse)

                        
n'est pas — différent de la valeur

                        
contient — utilise un joker *valeur*

                        
ne contient pas — exclut si la valeur est contenue

                    

                

                

                    
7 Sélecteur de valeurs

                    
En cliquant sur le champ valeur, une liste déroulante affiche les valeurs réelles présentes dans le cache AD. Vous pouvez :

                    
    
                        
  • Saisir du texte pour filtrer la liste
    • 
                        
  • Cliquer directement sur une valeur pour la sélectionner en un clic
    • 
                        
  • Cocher plusieurs valeurs pour créer plusieurs conditions d'un coup
    • 
                        
  • Cliquer sur Valider pour insérer toutes les valeurs cochées comme autant de lignes
    • 
                    

                

                

                    
8 Générer les CSV

                    
Une fois la règle enregistrée, utilisez le bouton Générer les CSV (ou l'icône téléchargement dans la carte) pour lancer la génération. Les fichiers sont créés dans application/output/, dans un sous-dossier horodaté.
