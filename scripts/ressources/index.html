Dossier de référence

La génération crée un nouveau dossier horodaté avec tous les CSV (historique).

Si la comparaison est activée, un delta est calculé entre les fichiers générés maintenant et le dossier sélectionné à gauche (clé : adresse mail). Il est écrit dans le sous-dossier __DELTA CSVs du nouveau dossier (même arborescence), chaque fichier listant les ajouts / suppressions (samAccountName;mail;type).
